Politique de Confidentialité — Gamoria

Dernière mise à jour : 13 juin 2026

Introduction

Gamoria ("nous", "notre", "l'application") est une application mobile de gestion de collection de jeux vidéo rétro développée par Damien Desanlis. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos données personnelles.

Données collectées

Données obligatoires

Compte utilisateur : - Adresse email (authentification) - Mot de passe (hashé, jamais stocké en clair) - ID utilisateur unique (généré automatiquement)

Collection de jeux : - Titres de jeux - Consoles associées - État de conservation (condition, boîte, manuel) - Notes personnelles (optionnel) - Prix estimés ou manuels (optionnel) - Photos personnalisées (optionnel)

Wishlist : - Jeux souhaités - Priorité (haute/moyenne/basse) - Prix cible (optionnel)

Matériel : - Consoles possédées - Accessoires - Éditions limitées

Données automatiques

Synchronisation : - Dates de création/modification des entrées - État de synchronisation (online/offline)

Préférences utilisateur : - Thème sélectionné (pixel/light/accessible) - Langue (FR/EN) - Devise préférée (EUR/USD/GBP/CAD) - Paramètres notifications prix

Données de prix eBay

Collectées via notre service : - Prix médians de marché (loose/CIB/new) - Historique des variations - Confiance du prix (high/medium/low)

Important : Ces données sont publiques (provenant d'eBay) et ne contiennent aucune information personnelle.

Données publicitaires (utilisateurs non Premium)

L'application affiche des publicités fournies par Google AdMob pour les utilisateurs de la version gratuite. À cette fin, Google et ses partenaires peuvent collecter :

  • L'identifiant publicitaire de l'appareil (IDFA sur iOS, AAID sur Android)
  • L'adresse IP et des informations techniques sur l'appareil (modèle, OS)
  • Des données d'interaction avec les annonces (affichage, clic)

Le niveau de collecte dépend de votre consentement (voir la section « Publicité » ci-dessous) : - Avec consentement : publicités personnalisées - Sans consentement : publicités non personnalisées (contextuelles uniquement)

Les abonnés Gamoria Premium ne voient aucune publicité et aucune donnée publicitaire n'est collectée les concernant.

Utilisation des données

Finalités

Authentification et sécurité : - Création et gestion de votre compte - Connexion sécurisée (email/password ou Google OAuth) - Isolation de vos données (vous seul y avez accès)

Fonctionnalités de l'application : - Sauvegarde de votre collection dans le cloud - Synchronisation multi-appareils - Suivi automatique des prix eBay - Génération de statistiques personnalisées - Export de vos données (CSV)

Amélioration du service : - Résolution de bugs (via logs anonymisés) - Optimisation des performances

Ce que nous NE faisons PAS

❌ Vendre vos données personnelles à des tiers
❌ Partager le contenu de votre collection avec des annonceurs
❌ Utiliser vos données de collection pour cibler des publicités
❌ Accéder à vos données sans votre consentement
❌ Analyser le contenu de vos collections à des fins commerciales

Publicité : l'application affiche des publicités via Google AdMob (utilisateurs gratuits). Ces publicités peuvent utiliser l'identifiant publicitaire de l'appareil selon votre choix de consentement, mais jamais le contenu de votre collection. Voir la section « Publicité ».

Stockage et sécurité

Infrastructure

Hébergement : Supabase (conforme RGPD) - Serveurs situés en Europe (région : eu-central-1 ou eu-west-1) - Chiffrement en transit (HTTPS/TLS 1.3) - Chiffrement au repos (AES-256)

Base de données : - PostgreSQL avec Row Level Security (RLS) - Isolation stricte par utilisateur (clause user_id = auth.uid()) - Pas d'accès administrateur aux données utilisateur sans raison valable

Stockage photos : - Supabase Storage (bucket game-photos) - Accès public aux images (URL directe pour affichage) - Isolation par dossier utilisateur ({user_id}/...)

Cache local

Stockage appareil (Hive) : - Cache temporaire pour mode offline - Non chiffré (données non sensibles : titres de jeux, prix publics) - Vidé automatiquement à la déconnexion - Aucune donnée sensible (pas de mot de passe, token, données bancaires)

Mesures de sécurité

✅ Authentification OAuth (Google)
✅ Sessions sécurisées avec refresh tokens automatiques
✅ Validation des entrées côté client et serveur
✅ Protection contre les injections SQL (Query Builder uniquement)
✅ Confirmations avant suppressions définitives
✅ Pas de partage de session entre utilisateurs

Cookies et tracking

Cookies fonctionnels : - Session d'authentification Supabase (JWT token) - Préférences locales (thème, langue)

Pas d'analytics tiers : - Pas de Google Analytics - Pas de Facebook Pixel

Publicité (utilisateurs gratuits) : - Google AdMob peut utiliser l'identifiant publicitaire de l'appareil et des technologies similaires, uniquement selon votre choix de consentement (voir la section « Publicité »). Aucune publicité ni collecte publicitaire pour les abonnés Premium.

Publicité

Fournisseur

L'application gratuite affiche des publicités via Google AdMob (politique de confidentialité de Google, fonctionnement des annonces Google).

Types de publicités

  • Bannières discrètes en bas de certains écrans
  • Annonces interstitielles occasionnelles (au maximum une toutes les 30 minutes, après plusieurs ajouts de jeux)

Votre consentement (RGPD / TCF)

Au premier lancement (dans l'Espace économique européen), un formulaire de consentement géré par la plateforme Google User Messaging Platform (UMP) vous est présenté :

  • Si vous acceptez : publicités personnalisées
  • Si vous refusez : publicités non personnalisées (basées uniquement sur le contexte, sans identifiant publicitaire de suivi)

Vous pouvez modifier ce choix à tout moment. Sur iOS, une demande App Tracking Transparency (ATT) peut également apparaître ; refuser le suivi limite la personnalisation des annonces.

Supprimer les publicités

L'abonnement Gamoria Premium désactive toute publicité et toute collecte de données publicitaire.

Partage de données

Avec qui partageons-nous vos données ?

Uniquement avec :

  1. Supabase (hébergement infrastructure) - Nécessaire pour le fonctionnement de l'app - Conforme RGPD, certifié ISO 27001 - Politique de confidentialité Supabase

  2. Google (si vous utilisez OAuth Google) - Authentification uniquement - Nous ne recevons que votre email - Politique de confidentialité Google

  3. eBay (prix de marché) - API publique sans transmission de données utilisateur - Requêtes anonymes via notre serveur - Pas de compte eBay requis

  4. Google AdMob (publicité — utilisateurs gratuits uniquement) - Affichage des publicités et mesure de performance - Peut recevoir l'identifiant publicitaire de l'appareil selon votre consentement - Ne reçoit jamais le contenu de votre collection ni votre email - Politique de confidentialité Google

Jamais avec : - Courtiers en données - Réseaux sociaux (sauf OAuth si vous choisissez) - Gouvernements (sauf obligation légale)

Aucun annonceur ne reçoit le contenu de votre collection ni vos données de compte. La seule donnée susceptible d'être partagée à des fins publicitaires est l'identifiant publicitaire de l'appareil, et uniquement avec votre accord.

Vos droits (RGPD)

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

1. Droit d'accès

Accéder à toutes vos données : - Directement via l'application (onglet Profil) - Export CSV (collection + wishlist) - Demande par email : damien.desanlis@gmail.com

2. Droit de rectification

Modifier vos données : - Édition directe dans l'app (jeux, consoles, profil) - Mise à jour en temps réel

3. Droit à l'effacement ("droit à l'oubli")

Supprimer votre compte : - Déconnexion → suppression locale immédiate - Suppression compte Supabase → toutes vos données sont effacées définitivement (cascade delete) - Délai : immédiat (données actives) + 30 jours (backups)

4. Droit à la portabilité

Exporter vos données : - Format CSV : Profil → "Exporter la collection" / "Exporter la wishlist" - Format JSON : disponible sur demande

5. Droit d'opposition

Refuser le traitement : - Notifications prix : désactivables dans Profil - Synchronisation cloud : mode offline disponible

6. Droit à la limitation

Geler temporairement vos données : - Contactez-nous : damien.desanlis@gmail.com
